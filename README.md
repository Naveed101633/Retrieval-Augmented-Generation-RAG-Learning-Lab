## 🚀 Retrieval-Augmented-Generation (RAG) Learning Lab

A practical, step-by-step lab to learn how modern RAG systems are built from scratch to production-level retrieval.

This repository is designed for learners, engineers, and AI builders who want to move beyond theory and actually implement RAG components with notebooks, experiments, and real-world patterns.

---

## 🎯 What You Will Learn

By completing this lab, you will be able to:

- Understand why plain LLM prompting fails for private, dynamic, or domain-specific data
- Build strong RAG fundamentals: retrieval, augmentation, generation
- Compare keyword search, semantic search, and hybrid retrieval
- Work with embeddings and vector similarity for semantic matching
- Measure retriever quality using Precision@K and Recall@K
- Apply chunking strategies for better context retrieval
- Understand query rewriting, filtering, and re-ranking in real pipelines
- Design production-minded retrieval flows used in real AI systems

---

## 🧭 Learning Path

| Module | Focus | Outcome |
|---|---|---|
| 01 - RAG Foundations | Prompt augmentation, LLM call patterns, grounded answering | Understand the core RAG idea and why retrieval comes first |
| 02 - Information Retrieval Foundations | TF-IDF, BM25, embeddings, semantic search, metrics | Learn how retrievers work and how to evaluate them |
| 03 - Retrieval with Vector Databases | ANN/HNSW, hybrid search, chunking, query parsing, re-ranking | Build scalable, production-style retrieval architecture |
